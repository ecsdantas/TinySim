Welcome to TinySim!

TinySim is a powerful simulation tool currently under development. While it may still have some bugs or incomplete features, it's ready for you to explore and test its capabilities!

To begin, click on "Library" and drag components onto the workspace. Connect the ports and run your simulation.

All wires must connect to ports
Output ports can have more than one connection.
Connections directly between wires are not supported.

General Tips:
  • Delete items: Select the item and press [delete] or [del] on your keyboard.
  • Flip horizontally: Select the item and press [I] on your keyboard.
  • Configure block: Select the block and press [O] on your keyboard, or click the gear icon.
  • Zoom: Use the scroll wheel to zoom in or out.
  • Move item: Click and hold the item, then drag it to the desired position.
  • Move several items: Click and drag to select multiple items, then drag any selected item to move all of them.
  • Undo/Redo: This feature is not supported in the current version.
  • Save project: Press [Ctrl+S] to save your current project.
  • Double-click on wire: Create an intermediate point that can be dragged to form more complex paths.

You can simulate up to 10,000 steps. All simulations and circuit data are processed entirely on your device. We do not store any circuit data on our servers.

Export to C Code

TinySim allows you to compile your project into C code, enabling further customization and integration into other systems. Keep the following in mind:

Important Considerations:
  • Always use a Display block as the output port for the generated C code.
  • Always use a Constant block as the input port for the generated C code.
  • All settings and parameters are copied into the generated C code.
  • Real-time synchronization is not supported in the current version.
  • The generated C code is divided into several files:
    • main.c: The entry point of your program. Customize this file as needed.
    • model.c: Contains the main model structure representing your diagram.
    • model.h: The header file for model.c, containing shared variables and functions.
    • libs.c: Standard TinySim functions related to diagram blocks.
    • libs.h: The header file for libs.c, containing shared variables and functions.
  • To compile the code, use: gcc *.c -o sim
  • To execute:
    • on windows: sim.exe
    • on linux: ./sim

TinySim is for desktop use only.
